# Teleporter Stations for VoxeLibre

A VoxeLibre mod that adds simple and immersive teleport mechanics.

## Mechanics

Place a Teleporter Base and name it. Use a (single-use) Teleport Core on any
station to teleport to any of the other stations around the world. The Teleport
GPS lets you view all stations and set waypoints toward them, so stations
function both as beacons as well as teleporters.

Only the station owner can break a station once placed.

## Compatibility

Tested with Luanti 5.15.2 on VoxeLibre 0.91.2. Should work with Mineclonia too.
